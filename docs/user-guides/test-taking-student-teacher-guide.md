### Setup Your Test Flow (5 minutes)

**Step 1: Create a Classroom**
1. Go to "Manage Classrooms" from dashboard
2. Enter classroom name (e.g., "Grade 3A")
3. Click "Create"
4. Copy the code (e.g., "ABC123") and share with students

**Step 2: Upload Vocabulary (Preparation)**
1. Go to "Manage Vocabulary"
2. Upload PDF or image with vocabulary words
3. Claude AI automatically:
   - Extracts vocabulary
   - Generates 3 test variants (A, B, C)
   - Creates questions (spelling, definition, multiple choice, fill-in-the-blank)
4. Wait for status to show "Completed" (2-5 minutes)

**Step 3: Assign Tests to Classroom**
1. Go to "Manage Classrooms"
2. Click "Manage" on your classroom
3. Click "Assign Test" button
4. Select test from available list (shows variant and question count)
5. Click to assign (you can assign multiple tests)

**Step 4: Monitor Students**
- See who's enrolled (students appear when they enter code)
- See which tests are assigned
- Remove tests if needed (click trash icon)

   - Click "Next Question" (or "Finish Test" on last question)

   - Progress bar at top
   - Question counter (e.g., "Question 3 of 10")
   - Must answer ALL questions before finishing (see warning if you miss any)

## Troubleshooting ğŸ”§

### "No Tests Assigned"
**Reason**: Teacher hasn't assigned any yet
**Solution**: 
- Ask your teacher if they uploaded vocabulary
- They need to generate tests and assign them to your classroom
- Check back in a few minutes

### "Can't Click 'Finish Test'"
**Reason**: You haven't answered all questions
**Solution**:
- You'll see warning: "You have X unanswered questions"
- Go back and answer missing questions
- Yellow warning box tells you exactly how many you missed

### "Got Answer Wrong But I Know It's Right"
**Reason**: Exact spelling required (case-insensitive)
**Examples of answers that are WRONG**:
- Typed: "apple" | Correct: "Apple" âœ“ (Case doesn't matter - will be marked correct)
- Typed: "appl" | Correct: "apple" âœ— (Spelling must be exact)
- Typed: "  apple  " | Correct: "apple" âœ“ (Extra spaces okay)

### "My Progress Disappeared"
**Reason**: Refreshed page or closed tab
**Note**: Test in progress needs to be completed - don't lose your work!
